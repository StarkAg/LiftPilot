# LiftPilot

LiftPilot is a rush-dispatch lift scheduler for crowded buildings. It uses a weighted cost function instead of AI to choose the best one-stop or two-stop plan, reducing painful upward stair walks while avoiding unnecessary lift stoppages.

Live link: https://liftpilot.gradex.bond

## Features

- Tablet-style tap interface for floors 1 to 15
- Cost-based stop selection with higher weight for walking up
- Live stoppage recommendations
- Expandable calculation view with split-by-split explanation
- Adjustable stair up and stair down cost sliders
